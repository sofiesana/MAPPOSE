# Multi-Agent Proximal Policy Optimization with Shared Experience (MAPPOSE)

Welcome to our repository for our implementation of MAPPOSE. This implementation was based on the RWARE gymnasium environmnent.

Files relating to the agent implementation can be found in the `agent` folder.

To run our simulation, please install the necessary packages from `requirements.txt`

Then, run: `python testing_out.py`

If you would like to run the version with reward shaping applied, run: `python testing_out_reward_shaping.py`
